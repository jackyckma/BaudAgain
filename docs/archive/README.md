## Archive Policy

Documents are moved to archive when:
1. They have been superseded by newer versions
2. The milestone/task they document is complete
3. The information has been consolidated into summary documents
4. They are no longer actively referenced

Archived documents are kept for:
- Historical reference
- Understanding project evolution
- Tracking architectural decisions over time
- Audit trail of completed work
